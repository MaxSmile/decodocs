# AI Bridge

Project site: https://ai-bridge.vasilkoff.com/

AI Bridge lets you choose or bring your own AI assistant and connect it to social channels (Facebook, Instagram, WhatsApp, Telegram) via AI Bridge connectivity. A public MCP is available on the site, and there is a demo entry point to get started quickly.

## Highlights
- Bring-your-own or select an AI chat and link it to preferred chat apps
- Social integrations: Facebook, Instagram, WhatsApp, Telegram
- Public MCP endpoint available for agents/integrations
- Demo flow (“Get started with DEMO”) to showcase the connection path
- Powered by Vasilkoff Team
