# SmartAIChats

Project site: https://smartaichats.com  
Vercel deploy: https://smartaichats.vercel.app  
DNS: Vercel recommends updating the `www` CNAME; root is 301 to Vercel, `www` currently 308.

AI-powered live chat widget offering a fast setup (“2 minutes to get started”) with a free widget flow.

## Highlights
- AI chat that responds to customer inquiries in real time
- Seamless site integration with minimal setup; includes “Get Free Widget” flow
- Fully customizable widget design
- Analytics integration: Google Analytics, Facebook Pixel, or other analytics
- Multilingual support
- Proactive chat triggers based on user behavior

## Product flow
- Collect email, send JavaScript snippet and auth link to customize widget settings
- All chat interactions routed to inbox by default
