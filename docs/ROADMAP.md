# Roadmap (Draft)

## Phase 0: Foundations
- confirm core decode workflow and terminology
- establish infra boundaries and security rules
- set up basic storage and auth integration

## Phase 1: Understand (Decode Core)
- check for mistakes and inconsistencies in docs
- translate and explain in plain language
- highlight risky or unclear sections with notes
- suggest alternative text using AI
- basic PDF editing

## Phase 2: Manage (Own the Document)
- secure file storage
- semantic search by meaning
- reminders tied to doc events (expiry, payment, send invoice)
- generate invoices and receipts
- templates for common agreements
- add watermarks

## Phase 3: Act (Share and Sign)
- send docs for signature via email and track status
- sign documents in-app
- generate a clean handwritten-style signature
- store signed versions and proof

## Phase 4: Integrations and Mobile
- HubSpot integration
- Google Drive integration
- Microsoft OneDrive integration
- iCloud integration
- iOS and Android apps (WebView first)

## Phase 5: Advanced Proof (Optional)
- electronic signature with wallet-based key (if needed)
- blockchain timestamping for document existence and integrity
